# Simple configuration for nvim

## Installation

1. Delete old nvim config:

```shell
rm -rf ~/.config/nvim
```

2. Install new config from this repo:

```shell
git clone https://github.com/borogk/nvim.git ~/.config/nvim
```

3. Run nvim:

```shell
nvim
```

## Nerd fonts

For proper icon display in the terminal download fonts from here https://github.com/ryanoasis/nerd-fonts
